Dijkstra's Algorithm Visualization

INTRODUCTION
Dijkstra's algorithm is a popular method used to find the shortest path between nodes in a graph with weights. Created by Edsger W. Dijkstra in 1956, it plays an important role in computer science, helping in things like navigation, internet routing, and managing resources. The algorithm works step by step, finding the best path from one starting node to others, as long as the weights are not negative.
Dijkstra's Algorithm is a foundational graph search algorithm used to compute the shortest path between nodes in a graph.
